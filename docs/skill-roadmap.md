## Implementation Guidelines

### For Each New Skill

1. **Research Phase** (30 min)
   - Review Microsoft documentation
   - Check community best practices
   - Identify common mistakes

2. **Writing Phase** (2-3 hours)
   - Create SKILL.md với 5-8 rules
   - Include correct/incorrect examples
   - Add integration examples

3. **Review Phase** (1 hour)
   - Self-review
   - Test code examples
   - Check formatting

4. **Enhancement Phase** (Optional, 1-2 hours)
   - Add PowerShell scripts
   - Create reference docs
   - Add to skill-runner

### Quality Checklist

- [ ] YAML frontmatter complete
- [ ] 5-8 rules with examples
- [ ] All code examples compile
- [ ] Integration example included
- [ ] Checklist provided
- [ ] References added
- [ ] Related skills linked
